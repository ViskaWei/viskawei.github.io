# Reinforcement Learning Framework for Stellar Target Selection in Galactic Archaeology


## 1. Introduction


This report outlines a reinforcement learning (RL) approach for optimizing stellar target selection in the field of galactic archaeology. The primary objective is to identify the oldest stars in the galaxy, with a particular focus on red giants characterized by very low surface gravity.


## 2. Reinforcement Learning Model


Given fixed time, 


### 2.1 State Space


The state space (S) is defined as the number of stars that have been processed at any given time step. This can be represented as:


S = {s | s ∈ ℕ, 0 ≤ s ≤ N}


Where N is the total number of stars in the available dataset.

### 2.2 Action Space
1. Look at which star and for how long  0,1,2….remaining hours


The action space (A) consists of the decision to select or not select a particular star for observation. This can be formalized as:


A = {a | a ∈ {0, 1}}


Where 0 represents not selecting a star, and 1 represents selecting a star for observation.


### 2.3 Reward Function: total number of times spent on valuable stars.


The reward function (R) is based on the scientific value of the selected star, with higher rewards given for older stars, particularly red giants with very low surface gravity. This can be expressed as:


R(s, a) = f(age, surface_gravity)


Where f is a function that quantifies the star's value based on its age and surface gravity.


### 2.4 Transition Function


The transition function (T) describes how the state changes after each action:


T(s, a, s') = P(s' | s, a)


Where s' is the next state, given the current state s and action a.


### 2.5 Policy


The policy (π) is the strategy the agent uses to select actions:


π(a | s) = P(a | s)


The goal is to learn an optimal policy π* that maximizes the expected cumulative reward.
